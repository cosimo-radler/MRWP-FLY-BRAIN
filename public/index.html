Brain Network Models Comparison

Interactive visualization of structural and robustness properties across different configuration models

Project Overview

This project compares various configuration models of Drosophila brain regions, analyzing their structural properties and robustness against both random failures and targeted attacks. The interactive dashboard allows for easy comparison between models, providing insights into how well different configuration models capture the properties of the original networks.

Brain Regions Studied

The project examines three key brain regions from the Drosophila connectome:

Ellipsoid Body (EB)

Part of the central complex involved in various locomotor behaviors and spatial navigation.

Fan-shaped Body (FB)

The largest structure of the central complex, playing roles in visual pattern memory and sleep regulation.

Mushroom Body Kenyon Cells (MB KC)

Key components of the mushroom body, essential for olfactory learning and memory formation.

Configuration Models Compared

Unscaled Config Model

Upscaled Config Model (3500 nodes)

Configuration model upscaled to 3500 nodes.

Clustering Config Model

Models that preserve both degree distribution and clustering coefficient.

Analysis Features

The dashboard includes multiple visualization tabs:

  • Degree Distribution: Log-log plots showing probability distributions of node connectivity
  • Random Percolation: Network resilience against random edge removal
  • Degree Centrality Attack: Network resilience against targeted removal of high-degree nodes
  • Betweenness Centrality Attack: Network resilience against targeted removal of high-betweenness nodes
  • Network Metrics: Comprehensive table of structural and robustness metrics

Toggle different models on/off to compare their properties and see how well various configuration models capture the structural and robustness characteristics of the original networks.
